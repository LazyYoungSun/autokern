# AutoKern CLI

Инструмент командной строки для автоматического расчета оптического кернинга шрифтов формата `.ttf` с использованием глубокого обучения (нейросетевой пайплайн на базе ансамбля моделей `ResNet18` и кастомной `MLP`-сети).

Утилита полностью изолирована и может вызываться из любой директории вашей операционной системы macOS.

---

## Требования к системе

Для установки и работы утилиты на вашем Mac должны быть:
1. Пакетный менеджер **Homebrew**.
2. Установленный менеджер пакетов **`uv`**.

---

## Установка

Откройте Терминал macOS и выполните следующие шаги:

### 1. Установите `uv` (если еще не установлен)
```bash
brew install uv
```

### 2. Клонируйте репозиторий и перейдите в папку проекта
```bash
git clone <ссылка_на_твой_репозиторий>
cd autokern
```

### 3. Установите утилиту глобально в систему
```bash
uv tool install . --force
```
`uv` сам создаст изолированную среду для скрипта, подтянет правильные версии тяжелых библиотек (`PyTorch`, `fontTools`, `Pillow`) под архитектуру вашего процессора и прокинет бинарник команды `autokern` в пути вашей macOS.

---

## Использование

Вы можете вызывать утилиту из любой папки компьютера, где лежат ваши рабочие шрифты. По умолчанию утилита автоматически подтягивает встроенную таблицу из 450 латинских пар, на которых обучалась модель.

### 1. Вызов справки
```bash
autokern --help
```

### 2. Базовый запуск
Утилита прочитает существующий кернинг из шрифта, добавит к нему усредненные предсказания ансамбля из 5 нейросетей и создаст новый файл с постфиксом `_kerned.ttf` в этой же папке:
```bash
autokern MyFont-Regular.ttf
```

### 3. Точечный докерн (флаг `-p`)
Если вам нужно быстро рассчитать кернинг и проверить модель всего на нескольких парах символов, передайте их строкой через запятую:
```bash
autokern MyFont-Regular.ttf -p "ьх,Го,AV,Te"
```

### 4. Запуск с кастомным списком пар (через файл JSON)
Если нужно протестировать модель на произвольном массиве пар, создайте файл `.json` в текущей директории и передайте его через флаг `-j` или `--json`:
```bash
autokern MyFont-Regular.ttf -j test_pairs.json
```
**Формат JSON-файла:**
```json
[
  ["Г", "о"],
  ["У", "а"],
  ["A", "V"]
]
```

### 5. Сохранение результата в конкретный файл
```bash
autokern MyFont-Regular.ttf -o ReadyFont.ttf
```
